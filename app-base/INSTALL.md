# Installation Guide

Follow these steps in order. Takes about 10 minutes.

---

## Step 1 — Unzip and open in VS Code

Unzip the file. Open the folder in VS Code.

---

## Step 2 — Install dependencies

Open the VS Code terminal (`Ctrl + `` ` ``) and run:

```bash
npm install
```

---

## Step 3 — Configure environment variables

1. Find the file `.env.local.example` in the project root
2. Duplicate it and rename the copy to `.env.local`
3. Open `.env.local` and fill in your Supabase values:

```
NEXT_PUBLIC_SUPABASE_URL=https://your-project-id.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
NEXT_PUBLIC_APP_NAME="Your App Name"
NEXT_PUBLIC_APP_TAGLINE="Your tagline here"
```

Get your Supabase URL and anon key from:
**Supabase Dashboard → Your Project → Project Settings → API**

---

## Step 5 — Start the app

```bash
npm run dev
```

Open your browser at: **http://localhost:3000**

---

## Step 6 — Deploy to Vercel

```bash
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/fm1960/your-repo-name.git
git push -u origin main
```

Then go to vercel.com → Import Git Repository → select your repo.
Add your environment variables in Vercel's project settings.
